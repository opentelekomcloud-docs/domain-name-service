Error Code
==========

Introduction
------------

When an API call encounters an error, an error structure is returned. The following table describes DNS error codes.

Error Code Structure Format
---------------------------

.. code-block::

   {
       "code": "DNS.0001",
       "message": "Internal error."
   }
